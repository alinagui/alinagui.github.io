Foreach语法

Foreach是Java SE5引入的一种新的更加简洁的for语法,主要是为数组和容器使用.foreach的作用跟for循环一样,但使用foreach语法将不需要创建int变量去对由访问项构成的序列进行计数,foreach将自动产生每一项.它的使用语法格式如下:

for(对象类型 输出值x : 对象)

如下代码:

public class ForEachFloat {

public static void main(String [] args){

Random random = new Random(47);

float f[] = new float[10];

for(int i = 0; i < 10; i++){

f[i] = random.nextFloat();

}

for(float x : f){

System.out.println(x);

}

System.out.println();

TextVector textVector = new TextVector();

textVector.showVector();

}

}

class TextVector{

Vector<Object> vector = new Vector<Object>();

public void showVector(){

for(int i = 1; i <= 5; i++){

vector.add(i * 10);

}

}

for(Object x : vector){

System.out.println(x);

}

}

Output:

0.72711575---0.39982635---0.5309454---0.0534122---0.16020656---

10---20---30---40---50---

*

程序中Random random = new Random(47)中的47作为种子,是伪随机数生成器的内部状态的初始值，随机产生的数字随着种子的改变而改变也就是使用了种子的随机数生成器其random对象在每一次运行程序时产生的数值都相同，而Random random = new Random()这中创建对象可以认为是默认使用时间作为种子.

代码中同样也使用了vector集合来测试foreach,这当然是可以的.ArrayList同样也行,说到ArrayList,可以为此注意的是,ArraList是Vector的新版,Vector是在Java1.0中就已经存在的,而ArrayList则是在Java1.2的时候出来的.相比较,ArrayList较新,但个人认为ArrayList并不是特别好.因为ArrayList不安全,虽然它速度快.而Vector安全,实在.所以一般用Vector,,,

再看程序,代码:for(float x : f)这条语句定义了一个float类型的变量x,继而将每一个f的元素赋值给x.Java中任何返回一个数组的方法都可以使用foreach,例如String类型的常量使用了toCharray()后返回一个char数组,这个时候就可以使用foreach方法来输出打印其中的字符了
